# Contributing to Mnemosyne

Thank you for your interest in contributing! Here's how to get started.

## Development Setup

```bash
git clone https://github.com/nianpangzhi233/Mnemosyne.git
cd Mnemosyne
pip install -e ".[dev]"
```

## Making Changes

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Ensure tests pass: `python -m unittest discover tests`
5. Commit with a clear message
6. Push and open a Pull Request

## Code Style

- Line length: 120 characters
- Python 3.10+ (use modern type hints)
- No unnecessary comments
- Follow existing patterns in the codebase

## Reporting Issues

- Use [GitHub Issues](https://github.com/nianpangzhi233/Mnemosyne/issues)
- Include OS, Python version, and steps to reproduce
- Tag with appropriate label (bug, enhancement, question)

## Questions?

Open an issue with the `question` label. We're happy to help.
