# Security Policy

## Supported Versions

| Version | Supported |
| ------- | --------- |
| 8.x     | Yes       |
| < 8.0   | No        |

## Reporting a Vulnerability

If you discover a security vulnerability in Mnemosyne, please report it by opening a [GitHub Issue](https://github.com/nianpangzhi233/Mnemosyne/issues) with the label `security`.

Please include:

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if you have one)

We will acknowledge your report within 48 hours and provide a more detailed response within 7 days.
